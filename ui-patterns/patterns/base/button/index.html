Button

A button element to trigger an action

<button class="base--button" type="button"> Click Me</button>

<button class="base--button" type="submit"> Submit</button>

<button class="base--button" type="reset"> Reset</button>

Button, <button>, is an element which may serve as a trigger to an action. Submit buttons are linked to forms and push the data from the form to the server. Reset buttons will clear any inputs.

Application Methods

  • .base--button -

    Add this class to an element

  • .base--STYLED button -

    <button> tag inside a class of .base--STYLED

  • %base--button -

    Extend this silent selector through Sass

Settings

  • secondary color -

    Background color of button

  • white -

    Text color of button
