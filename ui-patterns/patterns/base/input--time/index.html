Input - Time

Time input field for a form


<!-- {% pattern input from base %} -->
<label class="base--block-label" for="input1">Basic Time Input</label>
<input type="time" id="input1"   class="base--time-input" ></input>

<label class="base--block-label" for="input2">Placeholder Time</label>
<input type="time" id="input2" placeholder="This is placeholder time"   class="base--time-input" ></input>

<label class="base--block-label" for="input3">Disabled Time Input</label>
<input type="time" id="input3" placeholder="Can't click me"   class="base--time-input" disabled></input>
<!-- {% end pattern input from base %} -->

Basic time input for a form.

Application Methods

  • .base--time-input -

    Add this class to an element

  • .base--STYLED input[type="time"] -

    <input type="time"> tag inside a class of .base--STYLED

  • %base--time-input -

    Extend this silent selector through Sass

Settings

  • input padding -

    Padding of input box

  • input border width -

    Border width of input box

  • input border color -

    Color of input box border

  • background color -

    Background color of input box
