<label class="base--block-label" for="input1">Basic Datetime-local Input</label>

Basic datetime-local input for a form.

Application Methods

  • .base--datetime-local-input -

    Add this class to an element

  • .base--STYLED input[type="datetime-local"] -

    <input type="datetime-local"> tag inside a class of .base--STYLED

  • %base--datetime-local-input -

    Extend this silent selector through Sass

Settings

  • input padding -

    Padding of input box

  • input border width -

    Border width of input box

  • input border color -

    Color of input box border

  • background color -

    Background color of input box
