Textarea

Application Methods

  • .base--textarea -

    Add this class to an element

  • .base--STYLED textarea -

    <textarea> tag inside a class of .base--STYLED

  • %base--textarea -

    Extend this silent selector through Sass

Settings

    Padding of textarea

    Border width of textarea

    Color of textarea border

  • background color -

    Background color of textarea
