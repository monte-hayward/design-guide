Introduce Cognitive

Give first-time users a set of “cognitive training wheels” and continue to provide unobtrusive support as needed.

Provide intelligent defaults and positive feedback to reinforce cognitive behaviors and gradually adjust the user’s habits.

1.Based on the input data, this is a graph comparing REVENUE and TIME.
This describes the variables plotted on the graph.
2."Choose a different variable to plot"
This is copy that displays below a graph.

Clarify Watson’s capabilities with hierarchy, copy, and interactions to help set appropriate user expectations.

1."Remember that Chef Watson eats data, not real food. Use your own judgment when preparing these dishes."
This is copy that displays below the primary content.

Educate the user throughout the experience without interrupting their workflow.

1."You still need to complete an action to see results."
This is an empty module with copy that explains the user's next action.

Integrate additional explanation into the hierarchy as a secondary focus so that it doesn’t get in the way of more experienced users.

1."Did you know?..."
This is copy that displays beside the primary content.
